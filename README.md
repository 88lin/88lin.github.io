> 一个集成了节日特效、实用工具和娱乐功能的综合性导航网站

## 🌟 项目简介

茉灵导航是一个基于GitHub Pages搭建的综合性导航网站，旨在为用户提供丰富的网页特效、实用工具和娱乐功能。网站集成了多个精心设计的子项目，包括节日特效动画、音乐播放器、Windows 12网页版、书签管理器等，满足用户在不同场景下的需求。

  - [圣诞树二](Christmas2/index.html) - 发光圣诞树

  - [圣诞祝福](Christmas4/index.html) - 温馨圣诞祝福页面

### 🛠️ 实用工具
- **音乐播放器系统** - 基于Aplayer的在线音乐播放器
  - [音乐播放器](Music/index.html) - 支持多平台音乐源
  - 支持网易云音乐、QQ音乐、酷狗音乐等平台
  - 支持歌单、专辑、单曲播放
  - 键盘快捷键操作支持

- **Windows 12 网页版** - 模拟Windows 12操作系统的网页应用
  - [Windows 12](win12/desktop.html) - 完整的桌面体验
  - 支持深色/浅色主题切换
  - 包含计算器、设置等系统应用
  - 响应式设计，支持移动端访问

- **书签管理器** - 优雅的书签管理和导出工具
  - [书签管理器](Bookmarks/index.html) - 基于Pintree的书签系统
  - 支持浏览器书签导入导出
  - 美观的分类展示界面
  - 支持分享和协作功能

- **时钟与计时器工具** - 多样化的时间显示工具
  - [特效时钟](clock/index.html) - 动态时钟效果
  - [倒计时应用](https://timepulse.ravelloh.top/) - 功能丰富的倒计时器
  - [计时器](notion/8/index.html) - 简洁计时工具

## 🛠️ 技术栈

### 前端技术
- **HTML5** - 语义化页面结构
- **CSS3** - 现代化样式设计，包含动画效果
- **JavaScript** - 交互功能实现
- **jQuery** - DOM操作和事件处理
- **Bootstrap** - 响应式布局框架

### 构建工具
- **Webpack** - 模块打包和构建工具
- **Babel** - JavaScript编译器
- **Sass/SCSS** - CSS预处理器

### 特效库
- **Three.js** - 3D图形库（用于烟花、圣诞树等特效）
- **Aplayer** - 音乐播放器核心库
- **MetingJS** - 音乐平台API集成
- **Animate.css** - CSS动画库

### 部署平台
- **GitHub Pages** - 静态网站托管服务
- **CDN** - 静态资源加速

## 🚀 快速开始

### 本地开发
1. **克隆仓库**
   ```bash
   git clone https://github.com/88lin/88lin.github.io.git
   cd 88lin.github.io
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server
   ```

3. **访问项目**
   打开浏览器访问 `http://localhost:8000`

### 项目部署
项目已部署在GitHub Pages，直接访问：[https://88lin.github.io](https://88lin.github.io)

## 🎯 使用指南

### 导航使用
1. 访问主页后，可以看到分类清晰的导航按钮
2. 点击相应按钮即可跳转到对应的功能页面
3. 大部分页面支持移动端访问

### 特色功能使用
- **音乐播放器**：支持键盘快捷键操作，空格键播放/暂停，方向键控制音量和切换歌曲
- **Windows 12**：点击桌面图标体验完整的操作系统界面
- **书签管理器**：支持导入浏览器书签，自动生成美观的导航页面
- **古诗起名**：输入相关信息，系统会推荐适合的古诗词名称
