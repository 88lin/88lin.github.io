封面插图说明
• 推荐横向图片，宽高比 2:1 效果最佳

排版模式
Markdown 模式说明
• 使用 ### 创建小标题（每个小标题自动分页）
• 支持 **粗体**、*斜体*、列表、引用等格式
• 支持数学公式：行内 $E=mc^2$，块级 $$\sum_{i=1}^n x_i$$
• 支持代码块：```python 开始，``` 结束
长文模式说明
• 手动分页：使用分页标记控制换页位置
• --- 接续分页（下一页首段不缩进，适合段落中间分页）
• === 新段落分页（下一页首段缩进，适合另起新段落）
• 使用 # 一级标题、## 二级标题、### 三级标题
• 支持图片、公式、代码块、引用、列表等 Markdown 语法
• 适合论文摘要、读书笔记、长篇思考等场景
HTML模式说明
• 使用 <section> 标签分页，每个 section 生成一页
• 所有样式自动适配当前栏目主题色
• ⚠️ 正文页有 45px 左右页边距，50px 上边距，请注意内容宽度

插入图片到正文
图片使用说明
• 上传图片或输入URL后，点击"插入图片到正文"
• 上传的图片会显示为短标记 {{img:ID}}，便于编辑
• 支持 Markdown 模式和长文模式
• 图片会自动适配海报宽度
• 推荐上传本地图片，远程URL可能因跨域限制导出失败
• 上传的图片会自动压缩优化（最大宽度1200px，质量85%）
海报尺寸
比例说明
• 4:5 - 适合小红书、Instagram 等平台
• 3:4 - 适合微信公众号封面
• 9:16 - 适合手机全屏展示、抖音等短视频平台
正文字号
字号建议
• 内容较多时选择较小字号（15-16px）
• 内容精简时可选择较大字号（17-18px）
• 手机阅读建议 16px 及以上
导出海报
导出说明
• ZIP包：包含所有海报页面的高清PNG图片（每张约2160x2700像素）
• PDF文件：适合打印或分享完整文档
• 导出前请确保预览效果正确
• 远程图片可能因跨域限制导致导出失败，建议使用本地上传
